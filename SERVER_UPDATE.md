# Как обновить сервер

Изменения, которые мы сделали (исправление белого экрана), находятся **только на вашем компьютере**. Чтобы они заработали на сервере, вам нужно обновить его.

Вот что нужно сделать:

## Если вы используете Git (рекомендуется)
1.  **На компьютере**:
    ```bash
    git add .
    git commit -m "Fix ChefHat import error"
    git push
    ```

2.  **На сервере**:
    *   Зайдите на сервер через терминал.
    *   Перейдите в папку проекта.
    *   Скачайте обновления:
        ```bash
        git pull
        ```
    *   **Для Frontend (веб-сайта)**:
        Нужно пересобрать проект (если используете build) или перезапустить dev-режим.
        ```bash
        # Если используете npm run dev
        # Остановите процесс (Ctrl+C) и запустите снова:
        npm run dev
        
        # Если используете сборку (Production)
        npm run build
        # Перезапустите сервер, который раздает файлы (nginx/serve)
        ```
    *   **Для Бота/API** (если обновляли скрипты .py):
        ```bash
        # Если Docker:
        docker-compose down
        docker-compose up --build -d
        
        # Если просто Python:
        # Перезапустите процессы main.py и api.py
        ```

## Если вы просто копируете файлы
Вам нужно скопировать изменённый файл `src/app/components/RoleSelector.tsx` с вашего компьютера на сервер в ту же папку. После этого **перезапустить Frontend** на сервере.
